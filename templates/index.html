Instructions

Please fill out the form with the necessary information from the Statistics and Clustering pages in the CM Database.

  • Single Sites: Enter the number of single crew sites.
  • Double Sites: Enter the number of double crew sites.
  • Triple Sites: Enter the number of triple crew sites.
  • Car Clusters: Specify how many clusters of cars you need.
  • Van Clusters: Specify how many clusters of vans you need.
  • List of Bus Capacities: Provide a comma-separated list of bus capacities (in crews).

The Baseline Algorithm

  1. Submit the form using the numbers from the CM Database.
  2. Using your site map, place the minimum required number of crews — or as close as possible — in the buses.
  3. Resubmit the form after adjusting the site counts and removing the buses to reflect the crews you placed.
  4. If the result produces clusters that contradict the map, add a site to the buses, adjust the sites accordingly, and resubmit.
  5. Repeat the previous step as needed until the result aligns with your map.
If there need to be modifications based on site location, it will be a result of the Cluster subsection within Allocations. For more help, watch this video: https://youtu.be/8_hubfNbIc4
